## 1. 游戏概述

### 1.1 概念
你扮演德军集团军群总司令**龙德施泰德**，通过无线电文字指令指挥三位下属将军：**隆美尔、曼施坦因、古德里安**。将军们拥有 AI 自主权，会根据你的宏观指令和战场态势，自主做出战术决策，并以个性化风格向你汇报。

### 1.2 核心循环
```
输入指令 → AI 将军思考 → 战场更新 → 战报 → 点击下一回合
```

### 1.3 差异化卖点
- **不是 RTS，是"指挥官模拟器"** — 你控制战略，将军控制战术
- **AI 将军有性格** — 他们会争论、建议、甚至不服从
- **信息不对称** — 将军只知道自己视野内的战场
- **文字驱动** — 没有传统 UI 按钮，靠写信指挥

## 2. 角色系统

### 2.1 玩家 — 龙德施泰德 (Rundstedt)
- **身份:** 集团军群总司令
- **能力:** 下达宏观指令、分配目标、协调多路进攻
- **限制:** 不直接控制任何部队

### 2.2 AI 将军

| 将军 | 历史原型 | 性格关键词 | 行为倾向 | 典型回复风格 |
|------|----------|-----------|---------|-------------|
| 隆美尔 (Rommel) | "沙漠之狐" | 激进、大胆、亲临前线 | 冒险突进，经常冲太快，抱怨补给不足 | "我已抵达桥头，敌人正在溃退。给我补给，明天就能拿下全城。" |
| 曼施坦因 (Manstein) | 迂回大师 | 精密、冷静、战略家 | 喜欢迂回包抄，会提出复杂计划 | "正面强攻代价太大。建议我绕行北面树林，与古德里安形成钳形攻势。" |
| 古德里安 (Guderian) | 装甲战先驱 | 固执、专注、装甲至上 | 集中装甲力量突破，对步兵配合不满 | "没有足够的坦克燃料，我无法执行这个命令。" |

### 2.3 将军 AI 行为维度

每个将军每回合的行为由以下因素决定：
1. **收到的指令** — 玩家的文字命令
2. **战场视野** — 只有自己部队能看到的敌我信息
3. **性格倾向** — 影响决策风格
4. **当前状态** — 兵力、士气、补给
5. **历史记忆** — 记得之前的指令和承诺

## 3. 回合系统

### 3.1 回合流程

```
┌─────────────────────────────────────────────────┐
│ Phase 1: 战报                                    │
│ - 各将军汇报上回合结果                           │
│ - 显示战场变化                                    │
│ - 敌军活动简报                                    │
└────────────────────┬────────────────────────────┘
                     ▼
┌─────────────────────────────────────────────────┐
│ Phase 2: 你的回合                                │
│ - 文字输入框：选择目标将军 + 写指令              │
│ - 可选：附带态势描述（不附带则将军自己分析）      │
│ - 可以给多个将军分别下达指令                     │
│ - 点击"下达指令"发送                             │
└────────────────────┬────────────────────────────┘
                     ▼
┌─────────────────────────────────────────────────┐
│ Phase 3: AI 将军回合                             │
│ - 每个将军独立调用 LLM 生成回复                  │
│ - 回复包含：理解确认 + 行动计划 + 本回合执行内容 │
│ - 部队在网格上实际移动                           │
│ - 可能遭遇敌军触发战斗                           │
└────────────────────┬────────────────────────────┘
                     ▼
┌─────────────────────────────────────────────────┐
│ Phase 4: 敌军回合                                │
│ - AI 控制的敌方部队行动                          │
│ - 简单规则系统（或也可用 LLM）                   │
└────────────────────┬────────────────────────────┘
                     ▼
┌─────────────────────────────────────────────────┐
│ Phase 5: 结算检查                                │
│ - 是否达成胜利条件？                             │
│ - 是否触发失败条件？                             │
│ - 点击"下一回合"回到 Phase 1                    │
└─────────────────────────────────────────────────┘
```

### 3.2 回合时间
- **手动推进** — 玩家点击"下一回合"才继续
- 每回合 = 30 分钟游戏时间（或 1 小时，可配置）
- 总回合数：20-30 回合（一场战役 10-15 小时游戏时间）

## 4. 地图系统

### 4.1 网格
- **尺寸:** 16×12（A1-P12）
- **每个格子包含:** 地形类型、占领状态、是否有单位

### 4.2 地形类型
| 类型 | 通行 | 效果 | 视觉 |
|------|------|------|------|
| 平原 | ✅ 可通行 | 无加成 | 浅绿 |
| 树林 | ✅ 可通行 | 防御+20%，视野-1 | 深绿 |
| 河流 | ❌ 不可通过 | — | 蓝色 |
| 桥梁 | ✅ 可通行 | 关键目标 | 棕色 |
| 村庄 | ✅ 可通行 | 防御+30%，可驻扎 | 灰色 |
| 道路 | ✅ 可通行 | 移动速度+50% | 黄色 |
| 海滩 | ✅ 可通行 | 无加成 | 沙色 |
| 高地 | ✅ 可通行 | 视野+2，远程攻击+15% | 褐色 |

### 4.3 战争迷雾
- 每个将军的部队有视野范围（步兵2格，装甲3格）
- 将军只能看到自己视野内的格子
- 玩家（龙德施泰德）可以看到所有将军的视野并集
- 未探索区域显示为黑色

## 5. 部队系统

### 5.1 我方单位

**每个将军指挥以下单位：**

| 将军 | 单位 | 兵力 | 特殊能力 |
|------|------|------|----------|
| 隆美尔 | 装甲营 | 3辆坦克 | 突击+30%，燃料消耗快 |
| 隆美尔 | 步兵连 | 80人 | 占领据点 |
| 曼施坦因 | 步兵营 | 150人 | 防御+20%，工兵能力 |
| 曼施坦因 | 炮兵连 | 6门炮 | 远程火力支援 |
| 古德里安 | 装甲师 | 8辆坦克 | 突击+50%，需要补给 |
| 古德里安 | 摩托化步兵 | 100人 | 快速机动 |

### 5.2 单位属性
- **兵力:** 影响战斗力
- **士气:** 0-100，影响作战效率
- **补给:** 0-100，影响持续作战能力
- **疲劳:** 0-100，影响移动速度
- **位置:** 网格坐标

### 5.3 战斗公式
```
攻击力 = 兵力 × 士气系数 × 地形加成 × 随机(0.8-1.2)
防御力 = 兵力 × 防御地形加成 × 士气系数
伤亡 = 攻击力 / (攻击力 + 防御力) × 兵力
```

## 6. AI 将军系统

### 6.1 LLM 集成

**API:** 小米 API
**调用时机:** 每个将军每回合调用一次
**输入内容:**

```
[系统指令]
你是二战德军将军 {将军名}。你的性格：{性格描述}
当前回合：{回合数}
游戏时间：{时间}

[你的视野内战场态势]
{将军看到的地图数据}

[总司令的指令]
"{玩家输入}"

[你的部队状态]
{兵力/士气/补给/位置}

请回复：
1. 你对指令的理解
2. 你的行动计划
3. 本回合的具体行动（移动方向/目标坐标）
```

**输出格式:**
```json
{
  "understanding": "我理解总司令要我...",
  "plan": "我计划分两阶段执行：...",
  "action": "本回合我将向 C5 方向移动装甲营。",
  "movement": [{"unit": "装甲营", "targetX": 2, "targetY": 4}],
  "mood": "confident"
}
```

### 6.2 将军回复显示
- 以**无线电消息**风格显示在左侧面板
- 带将军头衔和时间戳
- 语气符合性格（隆美尔激进，曼施坦因精密，古德里安固执）

### 6.3 将军行为规则
- 如果没有收到指令，将军按上一回合计划继续行动
- 如果将军不同意指令，会提出替代方案
- 将军会主动汇报发现的敌情
- 补给不足时将军会请求补给

## 8. 胜负条件（暂定）

### 8.1 胜利条件
- **占领所有关键目标**（桥梁/村庄等标记点）
- 或 **歼灭敌方有生力量**（敌军兵力 < 20%）

### 8.2 失败条件
- **所有将军部队全军覆没**
- 或 **超时**（30 回合内未达成目标）
- 或 **关键目标被敌军占领**

### 8.3 评分
- **S 级:** 完美胜利（零将军伤亡）
- **A 级:** 速胜（< 15 回合）
- **B 级:** 惨胜（有将军部队损失过半）
- **C 级:** 险胜（勉强达标）
- **F 级:** 失败
